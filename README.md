# Mon Crochet Shop

Mon Crochet Shop là website bán sản phẩm handmade/crochet, gồm giao diện người dùng và hệ thống quản trị cho shop.

## Giới thiệu

Dự án hỗ trợ các chức năng cơ bản của một website thương mại điện tử:

- Xem danh sách và chi tiết sản phẩm
- Đăng ký, đăng nhập và quản lý tài khoản
- Thêm sản phẩm vào giỏ hàng
- Đặt hàng và xem lịch sử đơn hàng
- Áp dụng mã giảm giá
- Admin quản lý sản phẩm, đơn hàng, người dùng và voucher

## Công nghệ sử dụng

- Frontend: React, Vite, React Router, Axios
- Backend: Node.js, Express
- Database: MongoDB, Mongoose
- Authentication: JWT
- Upload ảnh: Cloudinary
- Email: SendGrid
